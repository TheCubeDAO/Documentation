Information

We recommend connecting using Java Edition Latest Version. Using other versions you may experience unexpected issues or bugs. 

There is no specific type of gameplay you have to abide by when playing on our servers, you can play however you like and do whatever you want & will still receive rewards as you play.

There is a player run auction house you can access with /ah to buy and sell items from other players directly with no risks.
You can set up a player warp or teleport to other players warps with /pw, view more info about this with /pw help. 
Automatic chest shops can be set up and you can become a merchant.

Instructions for Java users

1. In Minecraft Java Edition, go to Multiplayer, then Add Server
2. In the Server Address box, enter our IP address and click Done

Instructions for Windows Edition / Other Platforms

1. Go to Multiplayer, then Add Server - make sure your NAT type is open if on Console.
2. In the Server Address box, enter our Bedrock IP Address and click Done.
